## How to start

1. yarn install
2. yarn dev

## First time use it

1. yarn install
2. Install Prisma
3. Create Database (leave_bot)
4. Import dumb leave data into database
5. npx prisma db pull
6. npx prisma generate
7. yarn dev

## Machine Env

1. yarn --version > 1.22.19
2. npm --version > 8.19.2
3. node --version > v18.12.0
